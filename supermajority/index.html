Supermajority Calculator

This calculator determines the supermajority for a vote by summing the values assigned to each category (Support = 100, Neutral Leaning Support = 75, Neutral = 50, Neutral Leaning Oppose = 25, Oppose = 0), multiplying these values by the number of votes in each category, and dividing by the total number of votes. If the resulting supermajority is 60% or above, the vote passes; otherwise, it fails.
